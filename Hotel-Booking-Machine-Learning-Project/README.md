# Hotel Booking Machine Learning Project

## Overview

This project follows the complete lifecycle of a Machine Learning project using **Dataiku DSS**, from raw data cleaning to model deployment.

The objective is to transform raw hotel booking data into a reliable dataset suitable for predictive modeling. Throughout the project, I apply professional data preparation techniques, develop and evaluate Machine Learning models, and follow a structured workflow similar to those used in real-world Data Science projects.

---

## Project Objectives

- Clean and standardize raw data.
- Integrate multiple data sources.
- Prepare the dataset for Machine Learning.
- Prevent data leakage before model training.
- Train and evaluate predictive models.
- Build a complete end-to-end Machine Learning workflow using Dataiku DSS.

---

## Project Structure

### 01. Data Cleaning

Assessment and preprocessing of the raw datasets to improve data quality and create a consistent analytical dataset.

Main tasks include:

- Importing multiple datasets
- Handling missing values
- Correcting inconsistent data
- Transforming variables
- Merging external datasets
- Removing duplicates
- Aggregating business metrics
- Building the cleaned dataset

---

### 02. Data Preparation for Machine Learning

Preparation of the cleaned dataset before training Machine Learning models. During this stage, the dataset is transformed into a format suitable for predictive modeling by selecting relevant variables, removing features that could cause data leakage, encoding categorical variables, preparing numerical features and ensuring that the model learns only from information that would be available at prediction time.

Main tasks include:

- Defining the target variable
- Selecting relevant features
- Removing variables that may cause data leakage
- Encoding categorical variables
- Handling low-frequency categories
- Preparing numerical variables
- Building the final modeling dataset

---

### 03. Machine Learning

Development and training of Machine Learning models.

Main tasks include:

- Selecting suitable algorithms
- Training predictive models
- Hyperparameter tuning
- Comparing different models
- Interpreting model results

---

### 04. Model Validation & Evaluation

Evaluation of model performance using appropriate validation techniques and metrics.

Main tasks include:

- Model validation
- Performance comparison
- Confusion matrix
- Precision
- Recall
- F1-score
- ROC-AUC analysis

---

### 05. Deployment

Preparation of the final solution for production environments.

Possible tasks include:

- Workflow automation
- Model deployment
- Scheduled executions
- Performance monitoring
